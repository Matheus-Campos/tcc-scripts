# tcc-scripts

## Desenvolvimento

Crie um virtual env na raíz do projeto com o comando `python3 -m venv .`;

Ative o virtual env executando o script activate dentro da pasta bin. Comando: `source bin/activate`;

Instale as dependências com o comando `pip install -r requirements.txt`;

Crie uma pasta `data` na raíz do projeto e baixe os microdados do ENADE do site [https://www.gov.br/inep/pt-br/acesso-a-informacao/dados-abertos/microdados/enade](https://www.gov.br/inep/pt-br/acesso-a-informacao/dados-abertos/microdados/enade).
